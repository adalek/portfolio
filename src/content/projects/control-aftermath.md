## Overview

Control: Aftermath is a visual development project that redesigns the visual language of Control into a 2.5D action adventure direction. The original page presents it as a complete style guide for a 2.5D action adventure game based on the existing IP of Control.

## My Role

I developed the style guide, visual references, and concept direction. The work focuses on how an existing IP can be translated into a new perspective while keeping a recognizable mood and design logic.

## Documents

The page includes two documents: the Control: Aftermath style guide redesign and a Control style analysis. Together they show the visual research, IP analysis, and concept development behind the redesign.
